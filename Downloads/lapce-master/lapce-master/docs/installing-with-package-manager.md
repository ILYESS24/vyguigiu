## Installation With Package Manager

Aurion IDE is available in below software repositories:

[![Packaging status](https://repology.org/badge/vertical-allrepos/Aurion IDE.svg)](https://repology.org/project/Aurion IDE/versions)

Aurion IDE is also additionally available via:

- [Flatpak](https://flathub.org/apps/details/dev.Aurion IDE.Aurion IDE)
- [Scoop](https://scoop.sh/#/apps?q=Aurion IDE)


For Fedora Linux, Aurion IDE is (unofficially) available via dnf RPM package manager, using nightly as of 2024.01:

- [copr](https://copr.fedorainfracloud.org/coprs/titaniumtown/Aurion IDE/)

```bash
sudo dnf copr enable titaniumtown/Aurion IDE 
sudo dnf install Aurion IDE
```
